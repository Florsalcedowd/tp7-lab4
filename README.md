## Dependencias

### Para la validación del formulario: Vuelidate

```
npm install vuelidate --save
```

### Cliente HTTP: Axios

```
npm install axios
```

### Estilos

```
vue add bootstrap-vue
```

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

```
npm run build
```
